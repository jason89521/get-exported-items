# Get exported items

Get the exported items from a package.

## Install

```bash
npm install -g get-exported-items
```

## example

```bash
gei
# prompt
✔ Enter the package name … lodash-es
✔ Enter the folders that you want to search … .
# result
has default import: true

identity
isEqual
range
trimStart
isNull
isUndefined
pick
groupBy
...
```
